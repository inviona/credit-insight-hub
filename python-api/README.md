# Credit Default Risk - Python API

This FastAPI service serves your trained XGBoost model for credit default prediction.

## Setup

1. Copy your trained model artifacts into this folder:
   - `xgb_model.pkl`
   - `imputer.pkl`
   - `te_maps.pkl`
   - `feature_columns.json`

   These are generated by the notebook cells that call `joblib.dump()`.

2. Run locally:
   ```bash
   pip install -r requirements.txt
   uvicorn app:app --reload
   ```

3. Test:
   ```bash
   curl -X POST http://localhost:8000/predict \
     -H "Content-Type: application/json" \
     -d '{"AMT_INCOME_TOTAL": 250000, "AMT_CREDIT": 400000, "AMT_ANNUITY": 18000, "EXT_SOURCE_1": 0.75, "EXT_SOURCE_2": 0.72, "EXT_SOURCE_3": 0.80, "AGE_YEARS": 44, "YEARS_EMPLOYED": 14}'
   ```

## Deploy to Railway

1. Create a new Railway project
2. Connect your GitHub repo (or upload this folder)
3. Railway auto-detects the Dockerfile
4. Set environment variables if needed:
   - `GLOBAL_MEAN=0.0807` (training set default rate)
5. Get your deployment URL (e.g., `https://your-app.railway.app`)
6. Set that URL as `PREDICTION_API_URL` in your Lovable project secrets

## Deploy to Render

1. Create a new Web Service on Render
2. Connect your repo or upload via Docker
3. Set the start command: `uvicorn app:app --host 0.0.0.0 --port $PORT`
4. Get your URL and set it as `PREDICTION_API_URL`

## API Endpoints

- `GET /health` — Health check
- `POST /predict` — Single prediction
- `POST /predict/batch` — Batch predictions
